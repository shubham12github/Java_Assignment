What is Computer ?
Computer is a machine which accepts the information in digitised form and processes it in 
accordance with a variable programme, piece of software or set of instructions that 
specifies how the information would be handled. 

What is RAM?
RAM is volatile primary memory.  Data would be store in this memory while computer is 
running but vanishes it at a fraction of power off.

Where is data stored in computer?
In computer data is stored in Hard Disc Drive (HDD).

What is that input device used to type text and numbers on a document in the computer
